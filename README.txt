# iuto profile site

## ファイル構成

- index.html：プロフィールページ
- photos.html：写真ページ
- style.css：デザイン用CSS
- photos/：自分で撮った写真を入れるフォルダ

## 使い方

1. zipを解凍する
2. index.html をブラウザで開く
3. 写真を載せる場合は photos フォルダに photo-01.jpg などの名前で入れる
4. photos.html の画像パスを必要に応じて変更する
